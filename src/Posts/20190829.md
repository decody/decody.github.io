# Atom에서 Python 시작하기

#### 2019년 8월 29일

Atom 에디터에서 파이썬을 사용하기 위해서는 파이썬 패키지를 설치해야 한다. 패키지는 File > Settings 메뉴에서 Install Pakages에서 autocomplete-python과 script를 검색해서 설치한다.

파이썬 파일(*.py)를 생성하여 cntl+shift+b단축키를 누르면 자동 컴파일되어 Console 창에서 확인할 수 있다.

~~~python

print("Hello world")

~~~

컴파일 오류가 뜰 때는 Anaconda Prompt 창에서 해당 가상환경 활성화 여부를 확인한다.
